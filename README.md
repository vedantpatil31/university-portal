# university-portal
This project is a modern university portal designed for **Savitribai Phule Pune University – Department of Technology** using HTML, CSS, and JavaScript. It features a professional dashboard layout with animations, course information, login portals, admissions, reviews, and real-time integrations like Google Maps and WhatsApp. 

Savitribai Phule Pune University – Department of Technology Website

This project is a modern and interactive university portal designed for the **Department of Technology, Savitribai Phule Pune University**. The website is developed using **HTML, CSS, and JavaScript** and follows the design standards of professional university websites.
